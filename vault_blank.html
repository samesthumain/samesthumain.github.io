La bataille de Vault Blank

Une suite de courtes animations produites dans le cadre de mon cours de web. Basé sur un texte que j'ai écrit nommé "Humain de sang" sur des réfugiés devant se battre contre la tyrannie dans un monde fictif.

Tous les assets et le code ont étés faits par moi.

L'animation a été entièrement faite en JavaScript à l'aide du plugin "Jsap" et la vidéo a été capturée à l'aide de OBS.
